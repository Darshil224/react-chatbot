# 🤖 React Chatbot

A chatbot application built while learning **React** from scratch. This project was created as part of my React learning journey and focuses on understanding React fundamentals such as components, props, state management, controlled inputs, and dynamic rendering.

> **Note:** This is a learning project, so the code includes explanatory comments and intermediate implementations to document my understanding of React concepts.

---

## ✨ Features

- 💬 Send and display chat messages
- 🤖 Simulated chatbot responses using an external chatbot library
- ⚛️ Built using reusable React components
- 🔄 Dynamic rendering using `map()`
- 📦 State management with `useState`
- 🎯 Controlled input field
- 🆔 Unique keys using `crypto.randomUUID()`
- 🧩 Props for component communication
- 🏗️ Component-based architecture
